# Tic Tac Toe Game 🎮

A simple command-line Tic Tac Toe game built using Python as part of the Milestone Project in the "Complete Python Bootcamp" course by Jose Portilla.

## Features
- Two-player game
- Randomly selects which player goes first
- Player chooses X or O
- Input validation
- Prevents choosing occupied positions
- Detects winning conditions
- Detects tie games
- Option to replay after the game ends

## Concepts Used
- Functions
- Lists
- Loops (`while`, `for`)
- Conditional Statements
- User Input
- Boolean Logic
- Random Module

## How to Run
1. Clone this repository.
2. Open the notebook in Jupyter Notebook.
3. Run all cells.
4. Follow the on-screen instructions to play the game.
